# Sistema de Reserva de Cinema

Projeto Integrador desenvolvido para o programa Bolsa Futuro Digital.

## Objetivo

Desenvolver uma aplicação web para consulta de filmes, sessões e reserva de ingressos de cinema.

## Funcionalidades Previstas

* Cadastro e login de usuários;
* Listagem de filmes;
* Visualização de detalhes dos filmes;
* Consulta de horários;
* Reserva de assentos;
* Histórico de reservas;
* Interface responsiva.

## Tecnologias

* React
* React Router
* JavaScript
* HTML5
* CSS3
* Git e GitHub

## Status

Projeto em fase inicial de planejamento e desenvolvimento.
